1 Codex setup

  • Open Codex (the "Ask anything" window).
  • Open the boilerplate folder inside Codex (download it from the SharePoint link above first).
  • Set approval to "Approve for me."
  • Check the bottom row: project Capgemini Boilerplate, Work locally, branch main.

2 Run /kickoff

  • Type /kickoff → pick the Capgemini boilerplate.
  • Let it install and run. It opens a live preview — keep it open.

3 Build your Master Prompt

  • Open the Master Prompt builder: https://iq-capgemini-ai-crafting-lab.vercel.app/
  • Answer its questions: who it's for, the one job, the flow (open → do the thing → done).
  • Copy the Master Prompt it gives you.

4 Paste & first preview

  • Paste the Master Prompt into the Codex chat and send.
  • Let it build, then try the app in the preview.

5 Reiterate & build

  • Loop: see it → decide → ask in plain words (e.g. "Make the Save button bigger.").
  • Something off? Screenshot it, paste it, say what's wrong.
  • You direct, Codex builds. Keep going until you're happy.
