# AI Product Crafting Lab

Slide deck for the Capgemini x IQ "AI Product Crafting Lab" (Lab 2). A single self-contained `index.html` (19 slides, Capgemini-branded) plus image assets.

## View

Open `index.html` in any browser, or visit the live site:

https://ai-product-crafting-lab.vercel.app

Navigate with arrow keys / space / on-screen arrows / swipe. Press `F` for fullscreen.

## Deploy

```
vercel deploy --prod
```
